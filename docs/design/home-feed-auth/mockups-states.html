The flow, reused everywhere

AuthFlowCore steps
1 · Email — one field. Checks if the account exists (reuses checkUserExists).
2 · Password — existing email → log in. New email → set password.
3 · Onboarding — new accounts only. Invite link pre-fills the center, so onboarding skips that step.
4 · Resume — the pending action (RSVP, post, reply, join) completes automatically.
Hosts: sheet (mobile) · modal (desktop) · page (/auth deep links). Invite = same core, invite intent + validated code.
