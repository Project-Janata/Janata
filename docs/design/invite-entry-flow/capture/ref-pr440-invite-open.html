Decisions baked in:  •  First screen = the vouch banner + the reused onboarding intro (3 real slides), so a newcomer learns what Janata is before signing up.  •  "Verified" dropped — in a hard gate every account is verified, so the value is "you're in instantly," not a status word.  •  Center-agnostic (invite = Janata membership, not a specific center).  •  Signed-in users get a one-tap join; no/expired invite → #403's "paste a link / request one" gate.  •  Web fallback if the app isn't installed (most invitees).
